== Description ==

FS Product Inquiry is a WordPress product inquiry plugin. This plugin is useful for product inquiry and also store all the data in the database with email so we can use these emails as subscribe or for other product marketing purposes.

== Why Choose FS Product Inquiry? ==

== FS Product Inquiry is a great plugin for: ==

* Show all the products on Your Page
* You can identify how many users inquiry of your particular product by adding a name and email
* Also, you can see the inquiry details in admin backend -> FS Inquiries menu

== Shortcode ==
Use this [fspi-show-products-list] Shortcode to Show Inquiry Products.
* [fspi-show-products-list]

== Pro Features ==
